Forecast

React + Vite · Style-jsx · API OpenMeteo · Chakra

Aplicación web desarrollada con React + Vite que consume datos meteorológicos en tiempo real desde la API de OpenMeteo. Permite al usuario buscar ciudades, visualizar pronósticos detallados (temperatura, humedad, viento, etc.) y ofrece una interfaz responsiva y accesible construida con Chakra UI. Incluye filtrado dinámico de ciudades, desplegado en Vercel.

Banking Card App

React + Next.Js · Tailwind css · Postgre sql + Orm Prisma + Supabase

Se trata de un modelo demostrativo que representa un flujo de gestión de tarjetas bancarias. Su propósito no es ser funcional, sino ilustrar estilos, conceptos de diseño y la filosofía detrás de la interfaz. La intención es mostrar un posible recorrido de usuario y cómo podrían integrarse estas tecnologías en un escenario real.
